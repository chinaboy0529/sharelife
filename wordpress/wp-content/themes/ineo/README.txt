== Description ==

iNeo theme inspired by mg12's iNove theme.

Tested on WordPress 2.5.1, 2.6.1, 2.7.1, 2.8.3, 2.9.2, 3.0, 3.0.5, with IE 6/7/8/9, Firefox 2/3/3.5/3.6, Opera 9.6/10/11, Safari 4 and Chrome 2.0 up to 9.0.
Valid XHTML 1.1 and CSS 3!
Best view under 1024×768 or greater.
